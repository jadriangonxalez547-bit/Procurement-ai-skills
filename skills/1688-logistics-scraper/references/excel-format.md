# Excel 格式规范

## 文件输出

- 文件名：`1688待收货物流.xlsx`
- 路径：`d:\AI\1688待收货物流.xlsx`（脚本当前所在目录）
- 工作表名：`1688待收货物流`

## 7 列定义

| 列号 | 列名 | 宽度 | 对齐 | 备注 |
|------|------|------|------|------|
| A | 序号 | 6 | 居中 | 1, 2, 3... |
| B | 采购订单号 | 22 | 居中 | 19 位数字 |
| C | 订单状态 | 10 | 居中 | 带颜色高亮 |
| D | 物流名称 | 16 | 左对齐 | |
| E | 物流单号 | 22 | 居中 | |
| F | 物流更新时间 | 20 | 居中 | 格式 YYYY-MM-DD HH:MM:SS |
| G | 最新物流轨迹 | 60 | 左对齐 | 自动换行，最长 200 字符 |

## 表头样式

- 填充色：`#FF6B35`（橙色）
- 字体：白色加粗 11pt
- 对齐：居中
- 行高：28

## 状态颜色映射

| 状态值 | 颜色 | 含义 |
|--------|------|------|
| 已签收 | `#C6EFCE` 浅绿 | 快递已签收 |
| 派送中 | `#FFEB9C` 浅黄 | 快递员派送中 |
| 已揽件 | `#BDD7EE` 浅蓝 | 快递员已揽件 |
| 运输中 | `#FFC7CE` 浅红 | 快件运输途中 |
| 已发货 | `#BDD7EE` 浅蓝 | 商家已发货 |
| 待收货 | `#F2F2F2` 浅灰 | 订单级状态 |
| 物流异常提醒 | `#FF7F50` 珊瑚色 | 退回签收等异常 |
| 无 | `#FFFFFF` 白 | 无信息 |

## 边框

- 全部单元格：`thin` 灰色边框（`#CCCCCC`）

## 冻结与筛选

- 冻结窗格：`A2`（冻结表头）
- 自动筛选：`A1:G<最后行>`

## 行高

- 表头：28
- 数据行：50（保证轨迹自动换行后完整显示）

## 数据格式化规则

| 字段 | "无" 的含义 |
|------|------------|
| 订单号 | 未匹配到 19-22 位数字 |
| 状态 | `.logistics-status` 为空、且无关键词匹配 |
| 物流名称 | 第一行未匹配到"X快递：单号"格式 |
| 物流单号 | 同上 |
| 物流更新时间 | 无轨迹或无日期 |
| 最新物流轨迹 | 无轨迹事件 |

## 实现位置

`scripts/gen_excel.py` 中：
- 表头：第 7-15 行
- 状态颜色：第 19-29 行
- 列宽：第 80-87 行
- 行高：第 90-92 行
- 冻结与筛选：第 95-99 行
